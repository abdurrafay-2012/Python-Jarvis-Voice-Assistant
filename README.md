# Python Jarvis Voice Assistant

A simple and efficient voice assistant built with Python to automate basic daily tasks. This project focuses on understanding Voice-to-Text and Text-to-Speech logic.

## 🎙️ Features
- **Voice Control:** Opens Google and YouTube with simple voice commands.
- **System Automation:** Opens system apps like Notepad instantly.
- **Real-time Feedback:** Tells the current time and responds to user greetings.
- **Exit Logic:** Clean exit with "Stop" or "Exit" commands.

## 🛠️ Built With
- **Python:** Core programming.
- **SpeechRecognition:** To capture and process voice input.
- **pyttsx3:** For the assistant's voice output.

## 📝 How to Use
1. Install Python on your machine.
2. Run `pip install SpeechRecognition pyttsx3` in your terminal.
3. Run the `Jarvis.py` file and start speaking!
